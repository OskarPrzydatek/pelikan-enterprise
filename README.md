# Pelican business FE

Interface for Pelikan Holidays employees to create the offers.

## Run Project

After git clone

```
npm i
```

Run project

```
npm run dev
```

## Run Storybook - Components Documentation

```
npm run storybook
```

## Project CI

```
npm run test:coverage
npm run format
npm run lint:fix
```
